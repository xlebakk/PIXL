# PIXL — Image Editor

PIXL — это легкий и быстрый веб-редактор изображений, построенный на стеке Flask + OpenCV, с интеграцией локального ИИ через LM Studio и поддержкой компьютерного зрения.

## Основные возможности
* **Обработка изображений:** Поворот, отражение, изменение размера, фильтры (сепия, размытие, Canny, пороговая обработка).
* **Удаление фона:** Использование метода HED (Holistically-Nested Edge Detection) для чистого вырезания объектов.
* **AI-ассистент:** Интеграция с локальными моделями через LM Studio для анализа и генерации текстовых подсказок.
* **Камера:** Прямой видеопоток с веб-камеры с возможностью захвата кадра для дальнейшей обработки. (В разработке)
